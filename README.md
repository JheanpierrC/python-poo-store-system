# 🛒 Python OOP Store System

This project is a simple store system built using Object-Oriented Programming (OOP) in Python.

It was developed as part of a professional backend foundation roadmap, focusing on writing structured, maintainable, and scalable code instead of simple scripts.

---

## 🚀 Features

- Product creation with validation
- Stock management
- Shopping cart system
- Total price calculation
- Structured error handling (try/except)
- Type hints for better code clarity
- Clean separation of responsibilities

---

## 🧠 Concepts Applied

- Classes and Objects
- Encapsulation
- Basic Inheritance understanding
- Type Hints
- Exception Handling
- Separation of concerns
- Clean project structure

## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/JheanpierrC/poo-store-system.git


2. Navigate to the project folder:

cd poo-store-system


3. Run the program:

python app/main.py
